# 3.2 Brainstorming de Escenarios

El Brainstorming de Escenarios consiste en identificar y describir situaciones críticas que ponen a prueba los atributos de calidad del sistema. Cada escenario define un estímulo (evento que provoca una respuesta), el comportamiento esperado del sistema y su propósito dentro de la arquitectura de Nexus.
